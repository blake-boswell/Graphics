# Graphics

## What does it do?  
It outputs the vertices of resized letters! Woohoo!  

### File Input rules
The input must follow the formatting in the default "letters.txt" file. The formatting is as follows:
Number of letters

Letter 1  
Number of vertices  
Point 1  
...  
Point n  
  
Letter 2  
...  
  
Letter n  
Number of vertices  
Point 1  
...   
Point n
